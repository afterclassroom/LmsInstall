Lms Install
===================


Script for install Lms

----------


Get Lms install script
-------------

git clone https://github.com/afterclassroom/lms_install.git

Install
-------------

   

    $ cd lms_install

> Edit file webapp.conf change domain www.pesome.com with your Lms domain

Run file install.sh

    $ chmod +x install.sh
    $ sudo ./install.sh
